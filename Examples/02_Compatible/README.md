Description
-----------
This is an example for a native nodejs module using NAN.
The module is compatible with any other version of nodejs. 

Compatibility
------------

|Component|Version |
|-------|-----------|
|Ubuntu | 17.10     |
|node   | 6.11.4    |
|v8     | 5.1.281.108|
|gcc    | 7.2.0    |

Installation
------------
npm install

Run
---
npm test
