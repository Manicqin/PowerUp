Exmaple: 01_Basic
=================

Description
-----------
This is a minimal, most basic example for a native nodejs module.
The module's compatibility with the v8 api may break. 

Compatibility
------------

|Component|Version |
|-------|-----------|
|Ubuntu | 17.10     |
|node   | 6.11.4    |
|v8     | 5.1.281.108|
|gcc    | 7.2.0    |

Prerequisite
------------

g++ >= 7.2.0


Installation
------------
npm install

Run
---
npm test
